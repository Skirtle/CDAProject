To run multiplcation program: 
1. Compile and run main.c
2. First input must be a 1
3. Next two inputs are the numbers to multiply



IMPORTANT: PLEASE READ BEFORE RUNNING MAIN.C


To run the program begin by first compiling main.c

then run the .out or .exe from the command line folled by the name of the input file for the assembly code to run
For example: ./main.exe multiplication.txt or ./a.out multiplication.txt

Instructions for running multiplication.txt:
The program will ask for three numbers:
the first will be alway be one,
the second number will be the first number to multiply,
the third number will be the second number to multiply.

the output will be printed on the screen towards the end of the program's run.
